# 🚀 QUICK START: Deploy to Vercel in 5 Minutes

Your portfolio is ready to deploy! Follow these simple steps.

---

## ⚡ 5-Minute Setup

### Step 1: Create GitHub Repository (2 min)
1. Go to https://github.com/new
2. Name: `pragma-gtm`
3. Select "Public"
4. Click "Create repository"

### Step 2: Upload Files to GitHub (2 min)
1. Click "Add file" → "Upload files"
2. Drag all files from `pragma-gtm-github/` folder
3. Click "Commit changes"

### Step 3: Deploy to Vercel (1 min)
1. Go to https://vercel.com/new
2. Select your `pragma-gtm` repo
3. Click "Deploy"
4. **DONE!** Your site is live 🎉

You'll get a URL like: `https://pragma-gtm.vercel.app`

## 🌐 Add Custom Domain (Next: 10 min)

After Vercel deployment:

1. **In Vercel:** Settings → Domains → Add `pragma-gtm.com`
2. **In Namecheap:** Update nameservers to Vercel's
3. **Wait:** 24-48 hours for propagation
4. **Visit:** https://pragma-gtm.com ✅
